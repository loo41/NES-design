## Tips

<Text status="error">Input / Textarea share a common component, which needs to be distinguished by inputType</Text>
<Text status="error">输入框和文本框共用一个组件，需要使用 inputType 参数进行区分</Text>


## Case

<Playground wrapper="Input / TextArea">
  {
    () => {
      const [value, setValue] = useState('');
      return (
        <>
          <div style={{display: 'flex'}}>Input Value: <Text style={{marginLeft: 10}} status="success">{value}</Text></div>
          <Input
            value={value}
            onChange={e => setValue(e.target.value)}
            placeholder="Input"
          />
          <TextArea 
            value={value} 
            inputType="textarea" 
            placeholder="TextArea"
            onChange={e => setValue(e.target.value)}
          />
        </>
      )
    }
  }
</Playground>
